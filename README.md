KSAS Bulletin Board
=====================
**Current Version:** 5.0

Creates a custom post type for bulletins. Link to http://siteurl/bulletinboard/*slug* to display bulletins.

###Slugs
**Post-type slug:** bulletin-board  
**Taxonomy slug:** bbtype  
**bbtype terms:**  Undergraduate (undergrad-bb), Graduate (graduate-bb)

###Widget Controls
*	Title
*	Quantity
*	bbtype to display (dropdown choice)

###Admin tools
*	bbtype column
*	bbtype filter dropdown
